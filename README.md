# We Need to Talk About Limits

Resource management in the Cloud era.

## Presentation Stats

- **Duration:** ~20 minutes (optimized from 40+ minutes)
- **Focus:** Philosophy and principles over implementation details


## Key Themes

1. **The Problem:** Cloud promised unlimited resources but created new chaos
2. **The Principle:** Limits transform catastrophic failures into recoverable incidents
3. **The Practice:** Every layer needs explicit limits and graceful degradation
4. **The Philosophy:** Constraints enable reliability; unlimited resources are a myth
